CSS border-radius - Specify Each Corner

The CSS border-radius property defines the radius of an element's corners. Tip: This property allows you to add rounded corners to elements!

The border-radius property is actually a shorthand property

border-top-left-radius
border-top-right-radius
border-bottom-right-radius
border-bottom-left-radius

all in one border radius

elliptical corners:

border-radius:15px / 30px;

border-radius:30px / 15px;

border-radius:50%;

border-image property

border-image property allows you to specify an image to be used instead of the normal border around an element.

/* round the border */
border: 20px solid transparent;
border-image: url(border.png) 30 round;

/* it show red border when image is not visible */
border: 20px solid red;
border-image: url(border.png) 30 round;

/* stretch the border */
border: 20px solid red;
border-image: url(border.png) 30 stretch;

    background origin

    The CSS background-origin property specifies where the background image is positioned

             ________________________________________________________________________________________________
        |____________________________________..... background-origin ....._______________________________|
        |               |                                                                                |
        | border-box    |  the background image starts from the upper left corner of the border          |
        |               |                                                                                |
        | padding-box   |  the background image starts from the upper left corner of the padding edge    |   
        |               |                                                                                |
        | content-box   |  the background images starts from the upper left corner of the content        |
        |_______________|________________________________________________________________________________|

    background clip

    The CSS background-clip property specifies the painting area of the background.

             ________________________________________________________________________________________________
        |____________________________________..... background-clip ....._________________________________|
        |               |                                                                                |
        | border-box    |  the background is painted to the outside edge of the border                   |
        |               |                                                                                |
        | padding-box   |  the background is painted to the outside edge of the padding                  |   
        |               |                                                                                |
        | content-box   |  the background is painted within the content box                              |
        |_______________|________________________________________________________________________________|

    CSS Transitions

    CSS transitions allows you to change property values smoothly, over a given duration.

        
        transition-delay: 1s;
        transition-property: width;
        transition-duration: 2s;
        transition-timing-function: linear;
        
        transition-timing-function:
        ease: slow to fast
        linear: same speed from start to end
        ease-in: start with slow
        ease-out: end with slow
        ease-in-out: start to end slow transition

    CSS 2D Transforms

    CSS 2D transformations allow you to move, rotate, scale and skew elements.

            translate() function moves an element from its current position.
                 transform:translate(50px, 20px);
                
     
    
                
    

        rotate()function rotates an element clockwise or counter-clockwise according to a given degree.
                transform:rotate(60deg);
    
                
    
                
    
                
        scale() function increases or decreases the size of an element.
                transform:scale(1, 4)
                        
                                 Before applying scale:        After applying scale:

                                Width: 100px                   Width remains: 100px (1 × 100)
                                Height: 10px                   Height becomes: 40px (4 × 10)
                        
    
                
    
                
    

        scaleX() function increases or decreases the width of an element.
                transform:caleX(2);
                
    
                
    
 
        scaleY() function increases or decreases the height of an element.
                transform:scaleY(2);
    
                
    
                
    
        
        skew() function skews an element along the X and Y axis by the given angle.
                transform:skew(20deg, 10deg)
                transform:skew(20deg) (bydefault it will be take x axis)
                
                skew() function tlit the element
                        20deg → X-axis skew (horizontal tilt)
                        10deg → Y-axis skew (vertical tilt)
                
                
    
                
    

        skewX()function skews an element along the X axis by the given angle.
                transform:skewX(20deg);
                
    
                
    

        skewY() function skews an element along the Y axis by the given angle.
                transform:scaleY(20deg)
                
    
                
    

        matrix() function combines all the 2D transform functions into one.
                 matrix() function take six parameters are as: 
                              transform: matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())
                              transform: matrix(2, -0.3, 0, 1, 0, 0);
                              
                               _________________________________
                              |  a |  Horizontal scale (scaleX) |
                              |  b |  Skew Y                    |
                              |  c |  Skew X                    |
                              |  d |  Vertical scale (scaleY)   |
                              |  e |  Translate X (move right)  |
                              |  f |  Translate Y (move down)   |
                              |_________________________________|
